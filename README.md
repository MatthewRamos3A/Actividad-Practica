# Actividad-Practica
El programa es un sistema sencillo para organizar tareas hecho en Python. Funciona desde la consola y permite al usuario guardar, ver y eliminar tareas de manera fácil mediante un menú interactivo. La idea principal del programa es ayudar a practicar programación básica mientras se crea algo útil y fácil de entender.
Cuando el programa inicia, se crea una lista vacía donde se van guardando todas las tareas que el usuario escriba. En el menú principal aparecen las siguientes opciones:


Agregar tarea


Mostrar tareas


Eliminar tarea


Salir


La primera opción sirve para agregar tareas. El programa le pide al usuario que escriba una tarea, por ejemplo “hacer deberes” o “estudiar para el examen”, y luego la guarda en la lista. Después aparece un mensaje indicando que la tarea fue agregada correctamente.
La segunda opción permite mostrar todas las tareas que han sido guardadas. Si todavía no existe ninguna tarea, el programa avisa que no hay tareas registradas. Pero si ya existen tareas, estas se muestran en una lista numerada para que sea más fácil verlas y organizarlas.
La tercera opción sirve para eliminar tareas. Primero el programa muestra las tareas guardadas y luego el usuario escribe el número de la tarea que desea borrar. Después de eliminarla, el programa muestra un mensaje confirmando que la tarea fue eliminada correctamente.
El programa funciona continuamente gracias a un menú que se repite hasta que el usuario decide salir. Cuando se selecciona la opción 4, el programa termina su ejecución.
Este proyecto ayuda a practicar temas básicos de Python como listas, funciones, ciclos y menús interactivos, además de servir como práctica para subir proyectos a GitHub y aprender a organizar código de manera sencilla.
